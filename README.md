# industrial-ops-ui

Industrial web platform for multi-overhead-crane supervision.
Designed to replace traditional SCADA systems (WinCC, Ignition) with a modern,
real-time web interface accessible from any browser or mobile device.

## Stack
- Next.js 16.2.6 (App Router)
- React 19.2.4
- TypeScript 5
- TailwindCSS 4

## Modules
- Real-time overview (crane & tank synoptic)
- Recipe management
- Job queue
- Alarm management
- Reports (PDF / CSV)
- Administration (users & roles)

## Getting started
```bash
npm install
npm run dev
```

## Notes
This project is designed as a reusable foundation.
It can be adapted to any industrial environment
involving automated multi-line equipment.
